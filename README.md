Code and raw data created and used for the Cybercrime, Digital Traces and Forensic Data Analysis course (5274CDTF6Y) of the
Forensic Science MSc at the University of Amsterdam.

Consists of the following files:

1. detection_extraction.py
Used for extracting the first detection (if there is any) in all given trials

2. results cyber projectG1.csv
Holds the raw data before analysis. Combines both the data written down during tests as well as the extracted first detection times.

3. Cyber Project.Rmd
Used for running the statistical analysis.
